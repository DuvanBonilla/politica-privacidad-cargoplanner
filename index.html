Política de Privacidad

CargoPlanner es una aplicación desarrollada para optimizar la gestión de operaciones portuarias de forma eficiente y segura. A continuación, te informamos cómo tratamos los datos de los usuarios.

1. Uso de la información

La aplicación recopila únicamente los datos necesarios para el funcionamiento de las operaciones portuarias. Esto incluye información como nombre, cargo, ubicación y registros de tareas, los cuales son utilizados exclusivamente para fines operativos internos.

2. Permisos requeridos

  • Conexión a Internet: Se requiere para el correcto funcionamiento y sincronización de los datos en tiempo real.
  • Almacenamiento temporal: Algunos datos se almacenan de manera temporal en el dispositivo para garantizar la operación en zonas con conectividad limitada.

3. Modo Offline

La app permite trabajar sin conexión a internet. Los datos se conservan temporalmente en el dispositivo y se sincronizan cuando haya conexión disponible.

4. Compartición de datos

Los datos recolectados se comparten únicamente con las personas o entidades autorizadas por la empresa, y solo para fines relacionados con las operaciones portuarias.

5. Cambios en la política

Nos reservamos el derecho de actualizar esta política de privacidad. Cualquier cambio será notificado en esta misma página.

Si tienes preguntas o comentarios sobre esta política, puedes contactarnos al correo: tecnologiainnovacion@cargoban.com.co
